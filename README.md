# 👷‍♂️ Industrial-Safety-Detection-YOLO (基于 YOLOv5 的工业安全装备检测系统)

## 📖 项目概述
本项目旨在解决现代工业环境（如工地、车间）中传统人工监管效率低、易误判的痛点。通过深度学习目标检测技术，实现对工人安全装备（安全帽、反光背心等）的自动化实时监控，单次识别时间快，能够有效满足智能城市与工业管理的需求。

## ✨ 核心技术亮点 (Technical Highlights)

### 1. 强鲁棒性数据增强策略
针对工业现场复杂的光照、拍摄角度及背景干扰，本项目构建了全自动化的预处理流水线：
* **几何变换：** 引入随机旋转（30度内）、随机透视变换（20%失真度）以模拟不同监控视角。
* **色彩干扰：** 采用色彩抖动（Color Jitter）调节亮度、对比度和饱和度，模拟复杂光照环境。
* **泛化增强：** 结合随机灰度化处理，提升模型对不同环境色调的鲁棒性。

### 2. 优化训练策略
* **先进优化算法：** 弃用传统 SGD，采用 **AdamW 优化器** 结合权重衰减机制，有效抑制模型过拟合，增强训练过程的稳定性。
* **动态学习率调度：** 基于验证集表现动态调整学习率，避免模型陷入局部最优解，加速收敛并保证训练效果持续提升。

### 3. 多维度性能评估
系统通过一系列专业指标进行评估，确保工业级部署的可靠性：
* **定量分析：** 通过损失函数（Loss Curve）变化趋势监控模型收敛状态。
* **定性评估：** 利用 **混淆矩阵 (Confusion Matrix)** 深入分析不同类别（如安全帽 vs 普通帽子）之间的误判概率。
* **核心指标：** 关注精确率 (Precision)、召回率 (Recall) 及 F1-score，确保在工业红线指标上达到最优平衡。

## 🛠️ 技术栈
* **核心框架：** Python 3.x, PyTorch
* **视觉处理：** OpenCV, Pillow, Torchvision
* **工程辅助：** Matplotlib (可视化), Scikit-learn (指标评估)

## 🚀 快速启动
1. 克隆仓库。
2. 安装环境依赖：`pip install -r requirements.txt`。
3. 执行推理：`python detect.py --weights best.pt --source data/images/demo.jpg`。
